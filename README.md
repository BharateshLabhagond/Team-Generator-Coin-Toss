**Team Generator + Coin Toss**

Team Generator + Coin Toss web app that helps you quickly create teams for games.
You just enter the number of players, type their names, enter the number of teams, and the app will randomly split players into teams.

It also has:

* Shuffle button to reshuffle team members anytime
* Coin toss feature with a real flipping animation
* Duplicate name check while entering players
* Saved names list (click to reuse names easily)
* Reset button to clear everything and start again

The app is mobile-friendly and easy to use.
